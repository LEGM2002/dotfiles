# NoNameTheme

1. Clone the repository into your home

```bash
https://github.com/LEGM2002/dotfiles.git

# Using ssh (recommended)
git@github.com:LEGM2002/dotfiles.git
```

2. Change directory

```bash
cd dotfiles
```

3. Change install.sh permits

```bash
chmod +x install.sh
```

4. Run the installation script

```bash
./install.sh
```
